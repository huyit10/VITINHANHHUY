# Vi Tính Anh Huy

Website giới thiệu Vi Tính Anh Huy - Máy tính, linh kiện, thu cũ đổi mới.

## Công nghệ

- HTML5, CSS3
- Font: Be Vietnam Pro (Google Fonts)

## Cấu trúc

- `index.html` - Trang chủ (header, banner, sản phẩm nổi bật)
- `css.css` - Toàn bộ style

## Chạy local

Mở file `index.html` bằng trình duyệt hoặc dùng Live Server.
